# Nexora Gateway - Product Context

## Register
`product` — Admin dashboard, AI platform tool, Chat/Playground surfaces. Design serves the product.

## Users & Purpose
- **Platform admins**: Manage tenants, providers, endpoints, routes, billing. Dashboard-first.
- **Tenant operators**: Sub-platform operators managing their own users/providers/routes.
- **End users (developers)**: Create API keys, chat with models, test endpoints.

## Brand Personality
Professional, clear, technical. Modern AI platform feel without gimmicks.

## Anti-References
- Old-school admin templates (bloated menus, dated forms)
- AI-purple-gradient startup aesthetic
- Overly playful consumer-app feel

## Accessibility
Full keyboard nav, WCAG AA, dark mode by default (system preference aware).

## Tech
Vue 3 + TypeScript + Vite + Naive UI + Tailwind CSS
